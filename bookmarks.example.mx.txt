***bookmarks-mx***

### General Information ###

Bookmark line begins with "#  ". The "  ~  " separates information columns.

 #  50 Watts  ~  http://50watts.com/  ~  Illustration and book art with a literary bent. Focus on international illustrated books and Surrealism.  ~  visual arts, inspiration, illustration, archive
as
 #  website title  ~  address  ~  description  ~  tags

First two columns (website title, address) should be present, rest (description, tags) are up to the user, as it's simply intended to provide more releavnt information for user and more keywords for the fuzzy finder.

Contents of './save-bookmark-bookmarklet' (inline JavaScript) can be stored as a bookmark in web browser for quickly generating fzf-bookmarks text lines from the opened website data.
